Api documentation

Input File Format

File format is required to be .csv with MANDATORY columns:

  • ts - time of transaction (string/timestamp/integer)
  • user_id - unique identifier of user (string/integer)
  • item_id - unique identifier of item (string/integer)

Other columns will be considered additional features.

API

REQUEST:
HTTP GET:
URL: http://52.166.21.61:16346/api/recommend
JSON: ❴ "api_uid": str, "data": [[int]] ❵
  • api_uid - your personal id from account page
  • data - 2d list of lists of historical interactions for users
RESPONSE:
❴ "recommendations": [[str]], "error": str ❵
  • recommendations - 2d list of lists of recommended items for users from request
